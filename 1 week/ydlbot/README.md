# 🤖 AI-консультант фонда Шахмардана Есенова

Полноценный AI-консультант на базе **RAG**: Telegram-бот для пользователей + Streamlit
админ-панель. Отвечает строго по материалам сайта фонда, не выдумывает факты.

## Архитектура

```
Пользователь ──> Telegram-бот (aiogram) ──┐
                                          ├──> RAG ──> embeddings ──> FAISS (поиск)
Администратор ─> Streamlit (admin.py) ────┘                │
                                                           └──> Gemma 4 (генерация ответа)
Логи диалогов ──> SQLite (data/logs.db)
База знаний  ──> сайт фонда -> чанки -> FAISS (data/faiss.index + chunks.pkl)
```

| Слой            | Технология                                  |
|-----------------|---------------------------------------------|
| Telegram-бот    | aiogram 3                                   |
| Админ-панель    | Streamlit                                   |
| LLM (чат)       | Gemma 4 — `gemma4` через alem.ai            |
| Embeddings      | `text-1024` через alem.ai                   |
| Векторная база  | FAISS (`faiss-cpu`)                         |
| Логи диалогов   | SQLite                                      |
| Скрейпинг       | requests + BeautifulSoup                    |

## Запуск

**Шаг 1. Наполнить базу знаний** (скрейпинг сайта → эмбеддинги → FAISS):

```powershell
python -m app.ingest
```

**Шаг 2. Запустить Telegram-бота:**

```powershell
python bot.py
```

Откройте в Telegram **@yessenovaibot**, нажмите *Start* и задайте вопрос.

**Шаг 3. Запустить админ-панель** (в отдельном окне терминала, с активированным .venv):

```powershell
streamlit run admin.py
```

Откроется в браузере (обычно http://localhost:8501).

## LLM-провайдер

Используется **только alem.ai** (ключи из `creds.txt`), OpenAI-совместимый API:

- чат — `gemma4` → `https://llm.alem.ai/v1/chat/completions`
- эмбеддинги — `text-1024` → `https://llm.alem.ai/v1/embeddings`

Ключи и модели задаются в `.env` (секции `LLM_*` и `EMBED_*`).

## Гарантии заземления (anti-hallucination)

- Ответ генерируется **только** из найденного контекста (системный промпт в `app/rag.py`).
- Если релевантных чанков нет (score < `MIN_SCORE`) — бот отвечает дословно:
  **«Я не нашёл этой информации в материалах фонда»** даже не обращаясь к LLM.
- Модели явно запрещено придумывать дедлайны, суммы грантов, требования, даты и контакты.

## Полезные настройки (`.env`)

| Параметр          | Назначение                                        |
|-------------------|---------------------------------------------------|
| `SCRAPE_MAX_PAGES`| сколько страниц максимум обойти                   |
| `CHUNK_SIZE`      | размер чанка (символы)                            |
| `CHUNK_OVERLAP`   | перекрытие между чанками                          |
| `TOP_K`           | сколько чанков подаётся в контекст                |
| `MIN_SCORE`       | порог релевантности; ниже — «не найдено»          |
